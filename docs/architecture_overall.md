# Общая архитектура AI Компаньона

AI Компаньон — это «второй пилот» для игрока в динамичных играх (первая целевая игра — «Мир кораблей»).
Он анализирует экран в условно реальном времени, понимает контекст боя и даёт голосовые подсказки: приоритеты целей, тайминги перезарядки, риски засвета, рекомендации по манёврам.
Используется только на компьютере стримера создателя. Цель сделать геймплей более весёлым.

## Потоки данных (MVP)
1. Игра → Внутренний скриншотер Go → JPEG.
2. requester на GO: Скриншоты + SST (handy) → Промпт → OpenAI Plarform→ Ответ (текст/команды).
3. Ответ → api tts yandex → проигрывание в Go
4. Параллельно сообщения twitch → добавление в промпт
