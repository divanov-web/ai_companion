# Общая архитектура AI Компаньона

Коротко: система из 3 предварительных подсистем, соединённых простыми интерфейсами. Цель — развлекать игрока в "Мире кораблей".

## Подсистемы
- Скриншотер: делает снимки экрана каждые 2 с.
- Оркестратор (Go): собирает снимки, формирует промпт, отправляет в OpenAI, интерпретирует ответы.
- Озвучивание: преобразует ответы в речь и проигрывает.

## Потоки данных (MVP)
1. Игра → Скриншотер → PNG/JPEG.
2. Оркестратор: вход = Скриншоты → Промпт → OpenAI → Ответ(текст/команды).
3. Ответ → Озвучивание → Аудио в устройство вывода.

## Интерфейсы
- Storage(ScreenShot): Put(image), Get(lastN).
- LLMClient: Generate(context, prompt, images) → text.
- TTS: Speak(text) → audio.
