# Документация (подсказки для ИИ агента GoLand)

Коротко: используйте тезисный стиль и ссылочную навигацию.

## Быстрые промпты
- Создай/обнови краткий `README.md` для пакета, опиши назначение и публичные API (по пунктам).
- Сгенерируй схему интерфейсов домена (списком), без реализаций.
- Проверь соответствие кода идиомам Go 1.24 и предложи точечные правки.
- Сформируй список метрик/логов для ключевых операций и места их сбора.
- Обнови ссылки между файлами документации, не дублируя текст.
