# Requester (internal/app/requester)

Краткая справка по оркестратору CLI‑сценария «Послать запрос».

## Суть
- Собирает последние N изображений → обрабатывает → при необходимости создаёт диалог → отправляет одно сообщение с текстом и массивом изображений. Вся прикладная логика вынесена из `cmd/companion/main.go`.

## Ключевые особенности (важное)
- Выбор «последних» по времени изменения (mtime), а не по имени.
- Поддерживаются `.jpg/.jpeg` (регистронезависимо).
- Предобработка через `internal/service/image.Processor`:
  - ширина ≤ 1280 px, JPEG, качество ≥ 80;
  - целевой размер ≤ 1 МБ;
  - сохранение в `ImagesProcessedDir`.
- Очистка старых файлов в source и processed по TTL (секунды).
