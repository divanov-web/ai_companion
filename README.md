# AI Компаньон: Мир кораблей

Данная программа является частью программного обеспечения AI компаньона для совместной игры в "Мир кораблей".

### Важные понятия
- **AI Goland** — специализированный AI-помощник, помогающий в написании кода приложения на языке Go.
- **AI Компаньон** — разрабатываемая программа, предназначенная для добавления атмосферы игроку непосредственно в игровом процессе.

### Быстрый старт TTS (Yandex)
- Обязательный ключ: `YC_TTS_API_KEY`.
  - Как задать ключ (любой способ, приоритет снизу вверх):
    1) В `.env` в корне проекта: `YC_TTS_API_KEY=...`
    2) В переменной окружения ОС: `YC_TTS_API_KEY=...`
    3) Флагом запуска: `-yc-tts-api-key YOUR_KEY`
- Опциональные параметры: `YC_TTS_VOICE` (по умолчанию `ermil`), `YC_TTS_FORMAT` (по умолчанию `mp3`), `YC_TTS_SPEED` (по умолчанию `1.3` — ~на 30% быстрее), `YC_TTS_EMOTION` (по умолчанию `evil`).
- После получения текстового ответа ассистента приложение синтезирует и воспроизводит речь (mp3/wav). Параметры можно менять флагами `-yc-tts-voice`, `-yc-tts-format`, `-yc-tts-speed`, `-yc-tts-emotion` или переменными окружения.

Вспомогательные промпты для ИИ можно найти в [docs/README.md](docs/README.md).
